<h2>语法</h2> 
<p>top [-] [d delay] [q] [c] [S] [s] [i] [n] [b]</p> 
<h2>功能介绍</h2> 
<p>执行 top 命令可显示当前正在系统中执行的程序，并通过它所提供的界面，用功能键加以管理。</p> 
<h2>参数</h2> 
<p>d : 改变显示的更新速度，或是在交谈式指令列( interactive command)按 s</p> 
<p>q : 没有任何延迟的显示速度，如果使用者是有 superuser 的权限，则 top 将会以最高的优先序执行</p> 
<p>c : 切换显示模式，共有两种模式，一是只显示执行档的名称，另一种是显示完整的路径与名称</p> 
<p>S : 累积模式，会将己完成或消失的子行程 ( dead child process ) 的 CPU time 累积起来</p> 
<p>s : 安全模式，将交谈式指令取消, 避免潜在的危机</p> 
<p>i : 不显示任何闲置 (idle) 或无用 (zombie) 的行程</p> 
<p>n : 更新的次数，完成后将会退出 top</p> 
<p>b : 批次档模式，搭配 “n” 参数一起使用，可以用来将 top 的结果输出到档案内</p>
